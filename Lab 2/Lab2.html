    2. Verilog Programming using ISE

    Step 1: Opened terminal and typed ise to open the Xilinx ise Verilog IDE

    Step 2: Clicked Open Project...

    Step 3: Typed HelloProject into the Name field

    Step 4: Clicked Next and Next again then clicked Finish

    Step 5: Clicked Project

    Step 6: Clicked New Source

    Step 7: Selected Verilog Test Fixture and typed hello.v into the File Name field

    Step 8: Clicked Next and Next again and then Finish

    Step 9: Wrote the code for the hello.v file

    Step 10: Then clicked on Project then clicked on New Source

    Step 11: Selected Verilog Module and typed sum.v into the File Name field

    Step 12: Typed Sum in the Module Name field

    Step 13: Put a as the first Port Name and checked Bus and 15 under MSB and 0 under LSB

    Step 14: Put b as the second Port Name and checked Bus and 15 under MSB and 0 under LSB

    Step 15: Put s as the third Port Name and changed the direction to output and checked Bus and 15 under MSB and 0 under LSB

    Step 16: Clicked Next and then Finish

    Step 17: Wrote the code for sum.v

    Step 18: Under iSim Simulator, double-clicked Behavioral Check Syntax and then Simulate Behavioral Model to run the program

    Step 19: Finished

    hello.v:

    // Verilog Test Fixture Template

                  `timescale 1 ns / 1 ps
                
                  module hello_world;
                  //inputs
                 reg [15:0] a;
                 reg [15:0] b;
                  //outputs
                 wire [15:0] s;
                 Sum sumTest ( .a (a), .b (b), .s (s) );
                initial begin
                 a = 89;
                 b = 64;
                 $display ( "Hello Again!\n" );
                 $monitor ( "The sum of %d and %d is %d", a, b, s);
                 #80 $finish;
                    
                end
                
                  endmodule
